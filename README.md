## DesignMatch

DesignMatch is a minimalist product design studio and visual search tool built with React, Vite, TypeScript, Tailwind CSS, and Supabase.

### Getting started

- **Install dependencies**: `npm install`
- **Run locally**: `npm run dev`
- **Build for production**: `npm run build`

### Tech stack

- React + TypeScript
- Vite
- Tailwind CSS
- Supabase (Auth, Database, Edge Functions)

### Overview

Users can:

- Design on a mock product canvas with multi-side support
- Upload artwork (SVG/PNG/JPG) as editable layers
- Save and restore designs from Supabase with local fallback
- Run reverse-image product searches via Supabase edge functions

No third‑party builder metadata or tooling is required; everything runs as a standard React SPA.
